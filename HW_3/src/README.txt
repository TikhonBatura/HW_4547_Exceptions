Программа запускается из файла main.

Состоит из блоков:

    - control - модуль, где происходит обработка и анализ информации;
    - exceptions - модуль, где собраны потенциальные исключения и ошибки программы;
    - model - модуль, структуры искомых данных;
    - view - модуль, определяющий порядок действий получения, обработки и записи данных;

 Дополнительные комментарии находятся внутри классов.
